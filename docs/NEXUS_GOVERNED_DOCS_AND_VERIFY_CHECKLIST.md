# NEXUS Governed Docs and Verify Checklist

Use this checklist before publishing docs/automation changes.

- [ ] Scope documented in PR summary.
- [ ] Any runbook updates linked from affected docs.
- [ ] Registries updated when queue or watchdog behavior changes.
- [ ] No secrets, inbox payloads, or runtime logs included.
- [ ] `.gitignore` protects local-generated artifacts.
- [ ] Basic command checks executed and captured.
